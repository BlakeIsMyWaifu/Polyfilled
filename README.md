# Polyfilled

Polyfilled is an open source tech blog built with NextJS

[Preview](https://www.polyfilled.com)

## Requirements

- Node >= 16.x

## Installation

```md
# Clone repo
$ git clone https://github.com/BlakeIsMyWaifu/Polyfilled.git

# Install dependencies
$ pnpm i

# Rename example .env file
$ mv .env.example .env

# Build blog content
$ pnpm run build

# Run dev
$ pnpm run dev
```
